Bio

Hello, I'm Phillip. You'll know me as DonutDeflector online. I'm a hobbyist programmer, web developer, and computer nerd. I also enjoy playing the piano, drawing, and watching anime.

Web Development

My main focus is Front-End Web Developement. The combination of graphic design and logical problem solving really appeal to me. My approach to web design is KISS: Keep It Simple, Stupid. I aim to build fast, slim, and easy-to-use sites. Not everyone has access to fast internet and speedy hardware. I would like to allow these users to access my webpages quickly and without frustration.

Programming

I do a bit of programming on the side. I've programmed in C, C#, Javascript, Python, and Shell. My experience with C, C#, and Python is minimal. The bulk of my time is spent with Javascript and Shell.

Linux

I've been using Linux since 2014. Currently, my laptop runs Manjaro with XFCE and my desktop runs Arch with i3wm. My Raspberry Pi runs Rasbian and acts as a git server, seedbox, and RTMP server.
