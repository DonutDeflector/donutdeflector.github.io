adblocker

Year Q3 2016
Languages Shell, *Powershell
License MIT
Source Code Github

A collection of scripts that use /etc/hosts to block advertising-related, tracking, and malicious domains. Support for POSIX compatible shells and Windows Powershell.

Function

In short, adblocker takes these steps to block unwanted domains via the /etc/hosts file:

  1. backs up the /etc/hosts file as /etc/hosts.bk
  2. wgets StevenBlack's Unified Hosts Filter
  3. adds additional filters for reddit's tracking domains
This is all achieved in 23 lines of commented and formatted code.

Backstory

One of my friends on the interwebz talked about how he made his own little adblocker that utilized the /etc/hosts file to block advertizing/malvertizing/tracking domains. Pretty cool. He told me to make my own. Great. That's gonna turn out well.

Development

I had no idea how to do this. I looked at some tutorials and stumbled upon a Cup of Linux post. Ok, nice. Unfortunately, it didn't work for some reason. Or maybe I knew nothing about anything at the time. Right, so time to roll my own solution.

Couldn't be that hard, right? So, as my first forray into shell scripting, I wrote the script and made a post on the Linux Master Race subreddit. That gave me a few stars and two contributors. Not to bad.

This project was a pretty good learning experience, albeit a bit short. I was able to have my first experience working with contributors and learn more about git and GitHub. I also learned a few coding conventions regarding shell scripts. Overall, tons of fun.

* I didn't use this language myself, a contributor implemented a feature using the language.
